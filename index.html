Prompt Engineering

Specific prompts produce dramatically better results.

❌ Bad Prompt
"Build website"

2 words · zero context · AI guesses everything · generic output

✅ Good Prompt — In Depth
// 1. ROLE — tell AI who to be You are a senior full-stack developer specializing in React and AI integrations. // 2. GOAL — what to build & why Build an AI Resume Analyzer web app that helps job seekers improve their resume and pass ATS filters. Tech Stack • React 18 + Vite • Tailwind CSS for styling • Gemini API for resume analysis • pdf.js for client-side PDF text extraction • Deploy on Vercel Core Features 1. Upload resume — PDF only, max 5MB, drag-and-drop zone 2. Extract text from PDF using pdf.js (no server upload) 3. Send extracted text to Gemini API with this analysis: → ATS compatibility score (0–100) → Missing keywords for target job role → Weak bullet points + rewritten versions → Section-by-section feedback 4. Results dashboard with: → Large score ring + color-coded rating → Keyword gap chart → Suggestions rendered as markdown cards 5. Optional: match against a pasted job description 6. Download improvement report as PDF UI / UX Requirements • Clean dashboard layout — sidebar nav + main content • Dark theme with purple accent (like CareerOS) • Fully responsive — mobile, tablet, desktop • Loading skeletons while AI processes • Error states for failed uploads / API errors • Agent picker — HR, Technical, ATS perspectives Constraints • No backend for MVP — Gemini API called from frontend • Do not store resumes in a database • Handle API rate limits gracefully • Keep API key in .env — never hardcode Output Format Generate complete, working code — component by component. Start with project structure, then build page by page. Add brief comments explaining key decisions.

Skills vs Agents

Skills do one thing well. Agents combine many skills to finish a goal.

🔧
Skill
A single, focused capability — like a tool in a toolbox.
One-shot No memory Human-triggered Fixed scope
Examples
🌐
Translate
Convert text from one language to another.
📝
Summarize
Condense a long article into key bullet points.
🖼️
Generate Image
Create one image from a text description.
✉️
Write Email
Draft a single email with a given tone.
🔤
Fix Grammar
Correct spelling and grammar in a paragraph.
📄
Extract Data
Pull structured info from a PDF or webpage.
Best for: Quick, single-step tasks.
Limitation: Cannot plan, chain steps, or act on its own.
skills combine
🤖
Agent
Orchestrates many skills autonomously — like a junior developer.
Multi-step Has memory Self-directed Tool access

🔄

- Skills pills: hover scale effect, subtle border glow on hover

- Add a subtle divider or gradient line between sections

Prompt 4 of 6 — Add Gemini AI tagline generator
♥ Lovable — AI feature with Gemini
Add an AI-powered "Generate Tagline" feature to the Hero section.

Requirements:
1. Add a button below the tagline: "✦ Generate Tagline with AI"
2. On click, show a loading spinner
3. Call the Google Gemini API using the GEMINI_API_KEY secret I added in project settings
4. Use model: gemini-2.0-flash
5. Send this prompt to Gemini: "Write a professional one-line portfolio tagline for [MY NAME], a [MY ROLE] skilled in [MY TOP 3 SKILLS]. Maximum 15 words. Confident tone."
6. Display the AI-generated tagline below the button, replacing the static tagline
7. Add a "Regenerate" option to try again
8. Handle errors: show "Could not generate — try again" if API fails
9. NEVER expose the API key in frontend code — use a secure backend/edge function

The API key is stored as GEMINI_API_KEY in project secrets.
